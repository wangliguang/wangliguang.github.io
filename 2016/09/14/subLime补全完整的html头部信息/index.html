subLime补全完整的html头部信息

发表于 2016-09-14   |   分类于 H5   |  

前言

html有基本骨架，这个骨架能够用sublime快速生成，但sumlime生成的头部信息有时不是太完整，我们需要对sublime进行配置

安装package control

1、点击 Preferences > Browse Packages

2、找到Installed Packages文件夹

3、在下面网址下载文件

  https://sublime.wbond.net/Package%20Control.sublime-package

4、将下载的内容放到Installed Packages文件里

5、Restart Sublime Text

6、如果在 Preferences -> Package Settings中见到Package Control这一项，就说明安装成功了。

安装emmet插件

1、在链接中下载emmet plugin

  https://github.com/sergeche/emmet-sublime/archive/master.zip

2、打开sublime，preferences，browse packages，将下载的zip文件解压放到packages文件夹下，重启sublime

解决Sublime 3提示 Sublime Text Error while loading PyV8 binary

1、去gihub下载 PyV8,这个链接下面有各个平台的PyV8,记得下载相应平台的

  https://github.com/emmetio/pyv8-binaries#readme

2、打开sublime，preferences，browse packages,在packages文件中新建名为PyV8的文件夹，然后将下载的zip文件解压放到packages文件夹下，重启sublime

Dreamweaver CS6 Mac破解版
