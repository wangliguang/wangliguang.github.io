gitBook生成静态网页

发表于 2016-09-23   |   分类于 工具   |  

使用NPM安装gitBook

$ npm install gitbook-cli -g

创建一本书

$ gitbook init /你初始化的路径

编辑静态网址

首先需要让终端处于你init的路径

$ gitbook build

检验成果

在init的路径下会生成一个_book的文件夹，该文件夹内存放的便是生成的静态网页
